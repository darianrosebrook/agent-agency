## Kickoff

### Prerequisites

```bash
# Ensure you have:
- Node.js 18+
- npm or pnpm
- CAWS CLI installed
- Ollama running (for LLM inference)
```

### Initialize

```bash
cd iterations/bot-poc
npm install
npm run validate:all
npm run provenance:init
```

The arbiter should follow the build order in `docs/ARBITER_INSTRUCTIONS.md`:
